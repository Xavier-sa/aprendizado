Tabela de Funções JavaScript

Função Descrição
abs()Retorna o valor absoluto de um número
alert()Exibe uma janela com a mensagem repassada
confirm()Exibe uma janela e aguarda o usuário confirmar ou não
eval()Interpreta e executa strings com código JavaScript (⚠️ perigoso)
max()Retorna o maior valor numérico de dois ou mais
min()Retorna o menor valor numérico de dois ou mais
parseFloat()Converte uma string em número real
parseInt()Converte uma string em número inteiro
prompt()Exibe uma mensagem com campo de entrada
random()Retorna um número aleatório entre 0 e 1
round()Arredonda um número para o inteiro mais próximo
sqrt()Retorna a raiz quadrada de um número
String()Converte um valor para string (texto)
