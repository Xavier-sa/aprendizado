Comandos Git

Se você deseja atualizar sua branch com as últimas alterações da branch main, siga os passos abaixo:

git checkout main
git pull origin main

git checkout feature-branch
git merge main

# Resolver conflitos, se necessário:
git add .
git commit

# Enviar mudanças para o repositório remoto:
git push origin feature-branch

Para mais informações, visite o site oficial do Git.
